# IG — Portafolio de prácticas en Godot (GDScript & Shaders)

Repositorio de prácticas de **Informática Gráfica (IG)** desarrollado con **Godot**, centrado en programación de mecánicas, lógica de juego y trabajo con materiales/shaders.  
Este proyecto refleja mi evolución técnica a través de entregas iterativas (**p1 → p6**), aplicando buenas prácticas de scripting, organización modular y experimentación visual.

---

## 🎯 Objetivo del proyecto

Construir y consolidar competencias en desarrollo interactivo con Godot mediante prácticas progresivas, abordando:

- Arquitectura de escenas y nodos.
- Programación en **GDScript**.
- Integración de recursos gráficos y lógica de interacción.
- Uso de **GDShader** para efectos visuales.
- Iteración técnica con foco en calidad de entrega.

---

## 🧩 Stack técnico

- **Motor:** Godot
- **Lenguaje principal:** GDScript 
- **Shading:** GDShader 
- **Control de versiones:** Git + GitHub

El repositorio está organizado por prácticas:

## 🚀 Cómo ejecutar el proyecto

1. Clona el repositorio:
   ```bash
   git clone https://github.com/juakincruzz/IG.git
   ```
2. Abre Godot y selecciona la práctica que quieras revisar (`p1` a `p6`).
3. Importa el proyecto correspondiente y ejecútalo desde el editor.

> Recomendación: usar una versión de Godot compatible con la estructura y archivos `project.godot` de cada práctica.
